

#### 1\. Programming \& Languages

\----------------------------------------------

* #### Programming: Writing instructions for a computer to perform a specific task.
* #### Programming Language: A medium to communicate with computers and create software (e.g., Python, Java, C++).

#### &#x20;   Examples: ATM withdrawals, food delivery apps, and video recommendations.



#### 2\. Programming Paradigms

\--------------------------------------------

* #### Procedural Programming:

#### 

#### \*Focus: Functions, step-by-step logic, and sequence of steps.

#### \*Execution: Top-to-bottom.

#### \*Pros: Simple to learn, fast execution, suitable for small programs.

#### \*Cons: Low reusability, poor data security, difficult to maintain for large projects.

#### &#x20;   Examples: C, Pascal.

#### 

* #### Object-Oriented Programming (OOP):

#### 

#### \*Focus: Real-world entities represented as Classes and Objects.

#### \*Components: i) Attributes (properties) 

#### &#x20;            ii)Methods(actions).

#### \*Key Principles: Encapsulation, Inheritance, Polymorphism, Abstraction.

#### \*Pros: Highly reusable, better security, scalable, easier maintenance.

#### \*Cons: Uses more memory, slightly slower execution.

#### 

#### 3\. Overview of Python

\-------------------------------------

* #### Definition: A high-level, interpreted, general-purpose programming language emphasizing readability and simplicity.
* #### Creator: Guido van Rossum (1989; released in 1991).
* #### Key Features: Easy syntax, dynamically typed, open-source, portable, and huge library support.

#### 

#### 4\. Key Applications of Python

\---------------------------------------------------

#### \*Web Development: Django, Flask, FastAPI

#### \*Data Science \& AI: NumPy, Pandas, Matplotlib, TensorFlow, PyTorch

#### \*Automation \& Security: Web scraping, email automation, penetration testing

#### \*Game \& Desktop Apps: Pygame, Tkinter, PyQt

#### 

#### 5\. Memory Allocation Comparison

\-------------------------------------------------------

#### \*Procedural: Uses mostly Stack Memory to store variables and functions; minimal memory footprint.

#### \*OOP: Creates objects in Heap Memory with references in Stack Memory; uses more memory as each object stores its own attributes.
